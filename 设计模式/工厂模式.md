# 工厂模式

>>>
* 对new进行封装，使构造函数和创建者分离
* 可以根据需求创建不同的类
* 符合`开放封闭原则`(对扩展开放，对修改封闭)
>>>

```js
// 工厂类
class Creator {
  create (name) {
    return new Animal(name)
  }
}
// 工程方法
function createAnimal (name) {
  return new Animal(name)
}

// 要创建的类
class Animal {
  constructor (name) {
    this.name = name
  }
}

const creator = new Creator()

const duck = creator.create('Duck')
console.log(duck.name)

const chiken = create.create('Chicken')
console.log(chiken.name)
```
